PronouncUR: An Urdu Pronunciation Lexicon Generator

Home   |   Urdu Graphemes   |   Urdu Phonemes

This tool generates a pronunciation dictionary from a list of Urdu words in a form, suitable for use with a speech recognizer, such as CMUSphinx and text-to-speech system, such as MaryTTS.

Read my lips - this tool works only for Urdu. This means this tool only support Urdu graphemes. For the list of supported graphemes see this link. Urdu grapheme to phoneme mapping is explained on this link.

If you notice any errors in the output, such as a seemingly incorrect pronunciation, please report it and we will figure out what to do with it.

You can send reports to haris.zia@itu.edu.pk
